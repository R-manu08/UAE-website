# UAE Business Setup Website

A professional, high-conversion website for UAE business setup services, inspired by industry leaders Clara.co, Shuraa, and Virtuzone.

## Features

- **Elite Fintech Design**: Clean, modern interface with Deep Navy, Teal, and Gold color scheme
- **Interactive Quote Calculator**: 4-step process for instant cost estimates
- **Transparent Pricing**: Three package tiers with clear feature comparison
- **Mobile Responsive**: Optimized for all devices
- **Conversion Optimized**: WhatsApp integration, AI chat widget, contact forms

## Tech Stack

- HTML5
- CSS3 (Custom design system with CSS variables)
- Vanilla JavaScript (No frameworks)
- Google Fonts (Inter, Playfair Display)
- Font Awesome Icons

## Quick Start

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required!

## Customization

Update the following in `index.html`:
- Phone numbers (search for `971XXXXXXXXX`)
- Company name (currently "UAE Business Hub")
- Email addresses
- WhatsApp numbers

## Deployment

This is a static website that can be deployed to:
- Netlify
- Vercel
- GitHub Pages
- Any static hosting service
